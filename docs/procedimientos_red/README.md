# Procedimientos de Red

Aquí se documentarán los procedimientos estándar para la gestión y operación de la red del banco tecnológico.

## Índice

- [Configuración de Routers](#configuracion-de-routers)
- [Configuración de Switches](#configuracion-de-switches)
- [Gestión de Firewalls](#gestion-de-firewalls)

## Configuración de Routers

### Descripción
Los routers son dispositivos fundamentales en la infraestructura de red. Esta sección contiene procedimientos para su configuración, mantenimiento y troubleshooting.

**Documentos disponibles:**
- [Configuración de Router Cisco](configuracion-router.md)

### Pasos Generales
1. Acceder al dispositivo mediante consola o SSH
2. Ingresar al modo de configuración
3. Realizar los cambios necesarios
4. Guardar la configuración
5. Documentar los cambios realizados

## Configuración de Switches

### Descripción
Los switches son dispositivos de capa 2 que conectan múltiples dispositivos en la red local. Esta sección cubre su configuración y gestión.

### Procedimientos Comunes
- Configuración de VLANs
- Configuración de puertos troncales
- Configuración de STP (Spanning Tree Protocol)
- Gestión de MAC address tables

## Gestión de Firewalls

### Descripción
Los firewalls protegen la red de accesos no autorizados. Esta sección documenta las políticas y procedimientos de configuración.

### Procedimientos Comunes
- Configuración de reglas de entrada
- Configuración de reglas de salida
- Gestión de NAT (Network Address Translation)
- Configuración de VPN

## Notas Importantes

- Siempre crear un backup antes de realizar cambios
- Documentar todos los cambios realizados
- Realizar cambios durante ventanas de mantenimiento
- Comunicar cambios al equipo de soporte
- Verificar la configuración después de cada cambio
